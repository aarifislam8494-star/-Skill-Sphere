🎓 SkillSphere

A modern online learning platform designed to help discover and master new skills with ease. This platform features a premium user interface with smooth transitions and a seamless authentication experience.


🚀 Live Experience
View the Project: https://skill-sphere-mu-drab.vercel.app/


✨ Core Features

* Clean and responsive design for all devices
* Secure authentication with email and Google
* Real time course search and filtering
* Personalized student profiles with update features
* Modern hero slider with trending course highlights


🛠️ Technology Stack 

* Next.js 16 with App Router
* Better Auth with MongoDB Adapter
* Tailwind CSS 4 and DaisyUI 5
* React Toastify for notifications
* Swiper js for interactive sliders
